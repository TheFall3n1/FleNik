Welcome to Full Stack Web Development

A beginner's guide for HTML5.Learn all concepts of HTML5 (with hands-on assignments and projects.)


Are you a beginner? Do you want to learn web development but do not have any idea where to start?
If yes, then this is THE perfect course for you. In the HTML course, you will learn all the concepts of HTML5, starting from its history to its modern structure. We will cover all the tags which are most widely used in the world of web development. Simultaneously, we will show you some of the best practices of code-writing in HTML5.
HTML is short for 'HyperText Markup Language', which means HTML is only used to create a structure of the web page. But without some styling, no assignment or project will look good. Thus, we will cover some portion of the CSS required to create simple web pages.

This HTML course will be beneficial for all who are looking to start their career in the web development field and aspire to become a Frontend or Full-stack web developer. This HTML course will be the first step towards your goal to become a web developer.
To continue this journey, we would recommend you complete the other languages and skills like CSS, JavaScript, ES6, React. Check out our profile to find related courses that are designed just like this.

What is C Language?

C is a general-purpose programming language.


This language was developed in 1972 by Dennis Ritchie at the AT & T Bell Laboratories. It was known as C because most of its features were taken from an earlier language called 'B'. C is highly portable and is used for scripting system applications which form a major part of Windows, UNIX and Linux operating systems.

Basic Objective of this Course

This course is designed to provide complete knowledge of C language.


One will be able to develop logics which will help them to create programs and applications in C. Moreover, by learning the basic programming constructs, one can easily switch over to any other language in future.

List of Concepts to be covered

The main concepts involved in C language are as follows :-

  • Basic Programming Concepts
  • Operators and Conditions
  • Iterations
  • Functions
  • Arrays and Structures
  • Manipulating Arrays

Minimum Eligibility Criteria

No pre-requisites are needed.

This language requires no previous knowledge. Only basic knowledge of computer is required.
